説明

<script></script>
このタグの中ではJavaScriptの命令を使います、っていうHTMLタグ
document.write()
かっこの中身を画面（html）に出力しなさい、っていう命令
Math.floor()
かっこの中身の小数点以下を切り捨てなさい、っていう命令
Math.random()
0以上1未満の小数のうちコンピュータによって適当に選ばれた数（乱数）をください、っていう命令

つまり、0以上1未満の小数がランダムに出てきて、それに3をかけることで0以上3未満の小数にする。で、小数点以下を切り捨てることで0か1か2が得られる。それを画面（html）に出力する

じゃんけん

情報リテラシの授業で作ったフローチャートをもとにじゃんけんプログラムを作ってみましょう。

<script>
function janken(myhand) {
	var hand = ["ぐー","ちょき","ぱー"]; // 手の配列
	var aihand = Math.floor(3*Math.random()); // CPUが出す手を決める
	var result = ""; // 変数の初期化
	if( myhand == 0 ) { // 自分がぐーのとき
		if( aihand == 0 ) { // 相手もぐーなら
			result = "ひきわけ";
		} else if( aihand == 1 ) { // 相手がぐーでなくてちょきなら
			result = "かち";
		} else { // それら以外、つまり相手がぱーなら
			result = "まけ";
		}
	}
	// ここにelse ifを追加してじゃんけんを完成させよう
	// もちろんもっと簡単化してもOK

	// 結果表示
	alert("自分の手："+hand[myhand]+"\n相手の手："+hand[aihand]+"\n結果："+result);
}
</script>

<form>
<input type="button" value="ぐー" onclick="janken(0)">
<input type="button" value="ちょき" onclick="janken(1)">
<input type="button" value="ぱー" onclick="janken(2)">
</form>

ボタンじゃなくて画像にするとか、ポップアップで出すんじゃなくて相手の手も画像も変えるとか、いろいろ工夫できそうですね。
